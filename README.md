# Pachi's Resume Site

A personal one-page resume website for Pachi Parra, focused on developer relations, documentation, and developer experience.

## Short About

Personal resume website highlighting my Developer Relations work, writing approach, and experience helping teams improve developer onboarding and documentation.

## What's Included

- Intro and personal summary
- Work approach and priorities
- Professional experience overview
- Interests and personality section
- Contact links (email, LinkedIn, DEV.to)

## Tech

- HTML
- CSS

## Run Locally

1. Open `index.html` in your browser.
2. Or use a local server from this folder, for example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
